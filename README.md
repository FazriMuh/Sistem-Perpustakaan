## Installation

### Setting Up Virtual Environment (Optional)
```bash
python -m venv venv
source venv/bin/activate  # For Unix/Mac
venv\Scripts\activate  # For Windows
```

### Package Installation Code

To install all the required packages, use the following command:

```bash
pip install Flask
```
